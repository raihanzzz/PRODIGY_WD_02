# ⏱️ Modern Stopwatch Web App

A beautifully designed and feature-rich **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. This app provides accurate time tracking, lap recording, and a smooth user experience with modern UI effects.

---

## 🚀 Features

- ⏯️ Start / Pause functionality  
- 🔄 Reset stopwatch anytime  
- 🏁 Record lap times  
- ⌨️ Keyboard shortcuts support  
- 🎨 Modern glassmorphism UI  
- 📱 Fully responsive design  
- ⚡ High precision timing  

---

## 📂 Project Structure
```
📁 Stopwatch-App  
│── index.html  
│── style.css  
│── script.js  
```
---

## 🧠 How It Works

- Uses `performance.now()` for accurate timing  
- Updates every 10ms using `setInterval()`  
- Stores lap data dynamically  
- Uses class-based JavaScript structure  

---

## 🖥️ Interface

- Time display (00:00:00)  
- Buttons: Start, Pause, Reset, Lap  
- Lap history section  

---

## ⌨️ Keyboard Shortcuts

| Key        | Action        |
|------------|--------------|
| Space      | Start/Pause  |
| Ctrl + R   | Reset        |
| L          | Lap          |

---

## 🛠️ How to Run

1. Download or clone the repo  
2. Open `index.html` in browser
